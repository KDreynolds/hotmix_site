HoTMiX Documentation

A Comprehensive Guide to Getting Started with HoTMiX

Introduction

HoTMiX is a versatile command-line tool designed to streamline the process of starting new web projects. Leveraging the power of HTMX along with various backend technologies such as Flask, Gin, and Node.js, HoTMiX simplifies the initial setup and scaffolding of web applications. This tool is ideal for developers looking to reduce the time and effort in project initialization, offering easy scaffolding, diverse backend options, and seamless HTMX integration for a more efficient development process.

Installation

To install HoTMiX, you need to have NPM installed on your system. Type the following in your terminal:

npm install hotmixer --global

Getting Started

Once installed, you can start using HoTMiX to create new projects. Here's a quick guide:

  1. Open your terminal or command prompt.
  2. Run the command hotmix create myproject to create a new project.
  3. Select the backend technology when prompted.

This will create a new directory with all the necessary files to get started with your project.

Usage Examples

HoTMiX can be used in various scenarios, such as:

  • Creating a new web application with Flask and HTMX.
  • Setting up a Node.js project with pre-configured HTMX templates.

Example commands:

hotmix create blog --backend flask

This command sets up a new blog project using Flask as the backend.

HoTMiX supports the following backends:

  • Python - Flask
  • Golang - Gin
  • JavaScript - Express

Backend Technologies

HoTMiX supports several backend technologies, now including Lisp with Clack/Ten and Clack/Djula, alongside:

  • Flask: A lightweight and flexible Python web framework.
  • Django: A very popular Python web framework.
  • Gin: A high-performance Go web framework.
  • Echo: A minimalist Go web framework that provides a straightforward approach to RESTful API development.
  • Express: A fast, unopinionated, minimalist web framework for Node.js.
  • Koa: A new web framework designed by the team behind Express, aiming to be a smaller, more expressive, and more robust foundation for web applications and APIs.
  • Laravel: A PHP web application framework with expressive, elegant syntax, providing tools needed for large, robust applications.
  • Actix Web: A powerful, pragmatic, and extremely fast web framework for Rust.
  • Axum: A web application framework that focuses on ergonomics and modularity for Rust.
  • Mongoose: An embedded web server and networking library for C/C++ applications.
  • Lisp (Clack/Ten and Clack/Djula): Leveraging the power of Lisp for web development, Clack/Ten and Clack/Djula offer a robust and flexible framework for building web applications with Lisp. Clack is a web application environment that abstracts over various Lisp web servers, and Ten is a lightweight web framework built on top of Clack. Djula is a port of Django's templating engine to Common Lisp, allowing for easy and powerful web page generation.
